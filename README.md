# VPN MFA Checklist

Небольшая шпаргалка для первичной диагностики проблем с VPN и MFA.

Проект сделан как личный конспект: типовые ошибки, порядок проверки и шаблоны для передачи заявки дальше. Внутренние адреса, названия групп и реальные скриншоты из рабочих систем намеренно не добавлены.

## Что внутри

- `docs/vpn-checklist.md` — общий порядок проверки VPN
- `docs/mfa-push.md` — если не приходит push в приложении MFA
- `docs/tls-error.md` — ошибка SSL/TLS secure channel
- `docs/access-matrix.md` — обезличенный пример матрицы доступа
- `docs/escalation-template.md` — шаблон комментария при эскалации
- `scripts/find-sensitive.py` — простая проверка файлов на чувствительные строки

## Как пользоваться

Открыть нужный файл из папки `docs` и пройтись по пунктам.

Пример запуска проверки на чувствительные данные:

```bash
python scripts/find-sensitive.py
```

Скрипт не идеальный, но помогает быстро заметить, если в документации случайно остались URL, email или внутренние названия.

## Что не хранится в репозитории

- реальные домены и ссылки;
- имена внутренних групп;
- скриншоты из рабочих систем;
- ФИО, логины и email пользователей;
- точные названия серверов и ресурсов.

## Зачем это

Чтобы не искать каждый раз одно и то же в разных инструкциях и не держать в публичном репозитории лишнюю внутреннюю информацию.
