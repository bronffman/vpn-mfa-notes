# VPN MFA Checklist

Первичная диагностика проблем с VPN и MFA.

## Что внутри

- `docs/troubleshooting.md` — порядок проверки VPN-подключения, MFA и типовых ошибок
- `docs/tls-error.md` — диагностика ошибки SSL/TLS secure channel

## Что не хранится в репозитории

- реальные домены и ссылки;
- имена внутренних групп;
- скриншоты из рабочих систем;
- ФИО, логины и email пользователей;
- точные названия серверов и ресурсов.

## Notes

Внутренние адреса, названия групп и реальные скриншоты из рабочих систем намеренно не добавлены.
